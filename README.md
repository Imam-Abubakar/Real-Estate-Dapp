# Real Estate Dapp
 
 A Blockchain based real estate trading application. 
 
 Tech Stack used:
   1. Solidity
   2. React JS
   3. Hardhat
   4. EtherJS
   5. IPFS
   6. JavaScript
   7. Wallet - Metamask
 
This project requires node and hardhat framework installed.

For the transactions part use atleast 5 of the free accounts
